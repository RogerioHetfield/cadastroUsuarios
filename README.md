# Cadastro de Usuários

Projeto fullstack de cadastro de usuários desenvolvido com **React** no frontend e **Node.js** no backend, com integração a banco de dados **MongoDB** usando **Prisma**.

## Sobre o projeto

A aplicação permite cadastrar usuários e visualizar os dados em tempo real de forma online.

Esse projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais do desenvolvimento fullstack, incluindo:

- criação de interface com React
- consumo de API
- criação de rotas com Node.js e Express
- integração com banco de dados
- deploy do frontend e backend
- conexão entre frontend, backend e banco em produção

## Funcionalidades

- cadastrar usuários
- listar usuários cadastrados
- integração entre frontend e backend
- aplicação funcionando online
